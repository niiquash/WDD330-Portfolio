Useful Math Properties

  • Math.ceil - rounds up to the next integer. Eg: Math.ceil(4.2) >> 5
  • Math.floor - rounds down to the next integer. Eg: Math.floor(4.2) >> 4
  • Math.round - rounds to the nearest integer. Eg: Math.round(4.2) >> 4, Math.round(4.7) >> 5
  • Math.random() - returns and number between 0 and 1, both inclusive. Eg: Math.random() >> 0.78

Document Object Model

  • The DOM stores whitespaces as text.
  • All nodes in the dom have a numerical code to signify what type they are.
    • 1 - element
    • 2 - attribute
    • 3 - text
    • 8 - comment
    • 9 - body
  • Getting Elements from the DOM
    • getElementById()
    • getElementByTagName()
    • getElementByClassName()
  • Query Selectors are another way of selecting elements from the DOM
    • document.querySelector()
    • document.querySelectorAll()
  • Finding the value of a Node
    • .firstChild
    • .nodeValue
    • .textContent

Events

  • addEventListener() will attach an event listener to the document's body. It takes a parameter like 'click' and a function for the event.
  • Examples of mouse events are: click, mousedown, mouseup, dblclick, mouseover, mousemove
  • Examples of keyboard events are: keydown, keyup, keypress
  • An event can be removed from the dom by using the method '.removeEventListener()'.
